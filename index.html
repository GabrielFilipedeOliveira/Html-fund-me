Sobre o App

* Foi criado um frontend básico com o propósito de facilitar a interação com o SmartContact FundMe, cujo deploy foi feito na testnet Sepolia. Esse frontend utiliza informações essenciais, como o endereço do contrato, o ABI (Interface de Aplicação Binária) para interagir com as funções do contrato e um nó que se conecta à Metamask para garantir uma integração suave com a carteira dos usuários.

* A integração com a Metamask no frontend do SmartContact FundMe permite aos usuários conectarem suas carteiras Ethereum de forma segura ao aplicativo. Ao clicar para realizar ações, como contribuir para campanhas, a Metamask solicita permissões e exibe detalhes das transações, que os usuários devem assinar com suas chaves privadas. A Metamask encaminha as transações para a rede Ethereum, onde o contrato inteligente do FundMe as processa. O feedback é então exibido no frontend, facilitando a interação dos usuários com o contrato inteligente, garantindo segurança e controle sobre as transações. Isso torna a Metamask uma peça crucial na experiência de usuário para aplicativos baseados em Ethereum.
